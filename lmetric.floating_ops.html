Performs matrix multiplication using Triton and Torch libraries with FP16 data type.

Parameters:

  • sizes (list) – List of sizes for the matrices. Default is [256, 896, 1536, 2176, 2816, 3456, 4096].

  • metric (str) – Metric to measure performance. Valid options are ‘TFLOPS’ (default), ‘TBPS’, and ‘MS’.

  • kernels (list) – The kernels we are testing. Valid options are ‘triton’, ‘torch’

Returns:

None
